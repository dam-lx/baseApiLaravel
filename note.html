* Lỗi http bị chặn khi chuyển sang https

  • Khi bạn sử dụng SSL trong website Laravel của bạn. Khi đó, khi bạn gọi các hàm như asset để import JS hay CSS file thì mặc định nó trả về là http chứ không phải là https.
  • Vì vậy khi đó nếu bạn muốn khi sử dụng các hàm này thì buộc Laravel trả về giao thức https thì ta làm như sau: Mở file app/Providers/AppServiceProvider.php và cập nhật lại hàm boot như sau.
  • public function boot() { if(env('REDIRECT_HTTPS')==true) { \URL::forceScheme('https'); } }
  • Tiếp đó: Bạn mở file .env tại thư mục root của project rồi thêm 1 hằng số REDIRECT_HTTPS và thiết lập giá trị cho nó là true.
    REDIRECT_HTTPS=true
